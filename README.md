# 22.2.2-mon

Data Quality Monitoring App

## Server

### Install

1. [nodejs](https://nodejs.org/en/)
2. In this directory run
```shell
npm i
```
3. For external access use proxy. Locally install [ngrok](https://ngrok.com/download)

### Run

```shell
npm start
```
